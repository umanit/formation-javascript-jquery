Objectifs : afficher une carte Google Maps et positionner quelques markers.

Avec l'API Google Maps V3, et sans modifier le fichier HTML :

  1. Prendre connaissance de la documentation de l'API Google Maps V3.
  2. Créer une carte dont le centre se trouve en 47.18863349999999, -1.5519100999999864 (1 rue Victor Hugo à Rezé). Utiliser google.maps.Map.
  3. Écrire un script pour stocker les données du tableau HTML dans un tableau Javascript. La structure de données à obtenir est libre.
  4. Parcourir le tableau obtenu pour ajouter les restaurants à la carte, sous forme de markers. Utiliser google.maps.Marker.
  5. Centrer la carte sur les markers. Utiliser google.maps.LatLngBounds.
  6. Au click sur un des markers, ajouter la classe info à la ligne <tr>correspondante dans le tableau #restos. Utiliser l'attribut data-resto-identifier pour cibler la bonne ligne, et google.maps.event.addListener pour créer un écouteur d'évènement.
