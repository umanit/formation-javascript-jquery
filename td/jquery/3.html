Objectifs : intégrer des sons et vidéos. Développer un player vidéo en s'appuyant sur un plugin jQuery tiers.

En Javascript ou jQuery, sans modifier le fichier HTML :

  1. Jouer keyboard.mp3 à la saisie de texte dans le textarea, à chaque fois qu'une touche est pressée.
  2. Lorsque la touche Entrée est pressée, jouer carriage-return.mp3. Utiliser e.keyCode pour détecter la touche pressée.
  3. En Javascript ou jQuery, supprimer l'attribut controls de la balise <video>.
  4. Sur les boutons #play et #pause, brancher les méthodes play et pause de l'élément DOM <video>.
  5. Utiliser le plugin Slider de jQuery UI sur #volume (voir sa documentation). jQuery UI est déjà intégré à la page.
  6. Faire coulisser le slider doit modifier le volume de la vidéo. Agir sur la propriété volume de l'élément DOM <video>.
