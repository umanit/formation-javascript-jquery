Objectifs : créer un système d'onglets (tabs).

Avec jQuery, et la possibilité de modifier le fichier HTML :

  1. Ajouter des ancres pour lier sémantiquement menu et contenu.
  2. Un seul onglet ne peut être affiché à la fois. Utiliser une classe CSS pour cibler l'onglet ouvert.
  3. Le clic sur un des éléments du menu doit afficher l'onglet correspondant.
  4. Si une des ancres est présente dans l'URL au chargement de la page, afficher l'onglet correspondant.
  5. Créer un plugin jQuery pour encapsuler le code.
