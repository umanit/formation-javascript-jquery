Objectifs : créer un slideshow d'images.

Avec jQuery, et sans modifier (de préférence) le fichier HTML :

  1. Créer une fonction doSlide(index) permettant de déplacer la classe CSS active sur le slide identifié par le paramètre index.
  2. Appeler cette fonction au clic sur les boutons Précédent et Suivant (utiliser .filter() et .index() pour trouver le slide courant). Gérer les cas particuliers des premier et dernier slides à l'aide d'une condition if.
  3. Modifier cette fonction pour déplacer la classe CSS active de l'indicateur.
  4. Au clic sur un des indicateurs, appeler cette fonction pour afficher le slide correspondant (utiliser $(this) et .index() pour trouver l'index de l'indicateur).
  5. Créer un plugin jQuery pour encapsuler le code.
